# 导演账号功能客户端第四阶段派工指令包

## 1. 阶段目标

本阶段目标是打通“文字反馈为主体，绘制为增强层”的正式提交链路。

目标目录：`D:\project\MovtoolsEx\movtools-client\`

参考文档：

- `D:\project\MovtoolsEx\Doc\导演账号功能设计文档.md`
- `D:\project\MovtoolsEx\Doc\通信协议文档.md`
- `D:\project\MovtoolsEx\Doc\项目固化文档.md`

---

## 2. 本阶段必须完成

1. `保存为草稿` 生成当前镜头反馈列表项，不上传服务器
2. 列表支持编辑、删除、回跳到对应帧
3. 编辑反馈时，播放器跳回对应帧，并回填当前反馈说明
4. 允许只有文字反馈，不允许只有绘制提交
5. 若当前镜头只有绘制没有文字反馈，点击 `提交反馈` 时提示“尚未生成文字反馈”
6. 点击 `提交反馈` 时，按镜头上传：
   - 当前轮全部文字反馈
   - 当前轮全部本地绘制数据
   - 镜头状态
7. 点击 `镜头通过` 时，只上传镜头通过状态
8. 当前轮提交成功后，清空该镜头本地反馈列表与本地绘制数据
9. 提交失败时，本地反馈与本地绘制都必须保留，允许重试
