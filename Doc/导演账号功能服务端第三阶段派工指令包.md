# 导演账号功能服务端第三阶段派工指令包

## 1. 阶段目标

本阶段目标是让客户端能稳定回读“当前轮反馈 + 当前轮遮罩”。

目标目录：`D:\project\MovtoolsEx\movtools-server\`

参考文档：

- `D:\project\MovtoolsEx\Doc\导演账号功能设计文档.md`
- `D:\project\MovtoolsEx\Doc\通信协议文档.md`
- `D:\project\MovtoolsEx\Doc\项目固化文档.md`
- `D:\project\MovtoolsEx\Doc\镜头详情页设计文档.md`

---

## 2. 本阶段必须完成

1. `GET /api/review-feedbacks/lens/{lensId}` 或等价接口可按 `feedbackRoundId` 回读反馈卡片
2. 绘制数据回读接口可按镜头与当前轮回读单帧绘制集合
3. 镜头详情页回读或组合查询可明确给出当前镜头最近正式反馈轮次
4. 镜头详情遮罩只默认消费最近一轮正式 `feedbackRoundId` 对应的绘制数据
5. 不同轮反馈之间的数据不会混放成一套不可区分结果
6. 任务详情中的反馈计数、最近反馈时间、镜头状态统计保持一致
