# 第六批初始化验证说明

## 验证目标

- 客户端本地解析旧项目 Excel（含 `.xls` 与旧表头）
- 创建项目后再执行镜头批量同步
- 镜头同步成功后再创建本地镜头文件夹
- 页面能区分初始化失败、跳过、部分成功与全部成功

## 建议测试项

1. 使用旧项目真实 `.xls` 初始化文件创建项目
2. 分别模拟：未请求初始化、缺少 Excel、解析失败、镜头同步失败、文件夹创建失败、全部成功
3. 检查项目成员预置与成员管理是否仍可用
4. 使用 `maker` 账号确认仍可进入项目并编辑镜头

## 观察点

- 项目创建结果是否返回真实 `initResult`
- `pendingClientActions` 是否与实际客户端动作一致
- 本地镜头文件夹是否按真实镜头记录创建
- 页面是否把“项目创建成功”与“初始化全部成功”区分开
